# PRD: 심리테스트 웹서비스

## 1. 개요

### 1.1 프로젝트 목표

poomang.com을 벤치마킹한 모바일 최적화 심리테스트 웹서비스 구축

### 1.2 핵심 특징

- **비로그인 서비스**: 회원가입/로그인 없이 누구나 즉시 이용 가능
- **모바일 퍼스트**: 모바일 화면에 최적화된 반응형 UI
- **카드 기반 UI**: 심리테스트를 카드 섹션으로 시각적으로 표현

---

## 2. 사용자 요구사항

### 2.1 타겟 사용자

- 10~30대 모바일 사용자
- SNS를 통해 심리테스트를 공유하고 싶은 사용자
- 가볍게 즐길 수 있는 콘텐츠를 찾는 사용자

### 2.2 사용자 스토리

1. 사용자는 홈 화면에서 다양한 심리테스트를 카드 형태로 탐색할 수 있다
2. 사용자는 검색을 통해 원하는 테스트를 찾을 수 있다
3. 사용자는 테스트 상세 페이지에서 테스트 정보를 확인하고 시작할 수 있다
4. 사용자는 테스트 완료 후 결과를 SNS에 공유할 수 있다
5. 사용자는 좋아요/북마크를 통해 테스트에 반응할 수 있다

## 3. 기능 명세

### 3.1 홈 화면 (메인 페이지)

#### 3.1.1 헤더

| 요소 | 설명 |
|------|------|
| 로고 | 좌측 상단, 클릭 시 홈으로 이동 |
| 검색 아이콘 | 우측 상단, 검색 바 토글 |
| 햄버거 메뉴 | 좌측 상단, 네비게이션 메뉴 |

#### 3.1.2 검색 바

- 헤더 하단 또는 토글 형태로 표시
- 실시간 검색 또는 엔터 키 검색
- 검색어 기반 테스트 필터링

#### 3.1.3 테스트 카드 섹션

각 섹션은 테마별로 그룹화:

- 🔥 인기 테스트
- 🆕 최신 테스트
- 💕 연애 테스트
- 🧠 성격 테스트

#### 3.1.4 테스트 카드 컴포넌트

```
┌─────────────────────────┐
│   [썸네일 이미지]        │
│   (108px × 150px)       │
├─────────────────────────┤
│ 테스트 제목 (2줄 제한)   │
│ 👁 조회수  ❤️ 좋아요 수  │
└─────────────────────────┘
```

| 속성 | 설명 |
|------|------|
| 썸네일 | 테스트 대표 이미지 |
| 제목 | 2줄 말줄임 처리 |
| 조회수 | 테스트 실행 횟수 |
| 좋아요 | 좋아요 수 표시 |

### 3.2 테스트 상세 페이지

#### 3.2.1 상단 영역

```
┌─────────────────────────────┐
│     [테스트 대표 이미지]      │
│        (전체 너비)           │
├─────────────────────────────┤
│ 테스트 제목                  │
│ 테스트 설명 (2~3줄)          │
├─────────────────────────────┤
│ ⏱ 예상 소요시간: 3분         │
│ 👁 실행 횟수: 12,345회       │
├─────────────────────────────┤
│    [ 🚀 테스트 시작하기 ]     │
├─────────────────────────────┤
│ ❤️ 좋아요  🔖 북마크  📤 공유 │
│       👍 추천해요            │
└─────────────────────────────┘
```

#### 3.2.2 상세 정보 항목

| 항목 | 설명 |
|------|------|
| 대표 이미지 | 테스트를 대표하는 이미지 |
| 제목 | 테스트 이름 |
| 설명 | 테스트에 대한 간단한 소개 |
| 예상 소요시간 | 테스트 완료까지 걸리는 시간 |
| 실행 횟수 | 테스트가 실행된 총 횟수 |
| 테스트 시작 버튼 | CTA 버튼, 클릭 시 테스트 시작 |

#### 3.2.3 인터랙션 버튼

| 버튼 | 기능 |
|------|------|
| 좋아요 ❤️ | 테스트에 좋아요 표시 (로컬 저장) |
| 북마크 🔖 | 나중에 보기 위해 저장 (로컬 저장) |
| 공유하기 📤 | 카카오톡, 트위터, 링크 복사 |
| 추천해요 👍 | 추천 수 증가 |

#### 3.2.4 하단 영역 - 비슷한 테스트 추천

```
┌─────────────────────────────┐
│ 비슷한 테스트               │
├─────────────────────────────┤
│ [카드] [카드] [카드] →      │
│ (가로 스크롤)               │
└─────────────────────────────┘
```
